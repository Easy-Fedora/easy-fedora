# Оновлення системи

Для повного оновлення системи виконуємо команду в терміналі.

```bash
sudo dnf upgrade --refresh
```

{% hint style="warning" %}
Тим не менше, команда Fedora Linux рекомендує оновлюватися штатним способом через магазин додатків Gnome Software.
{% endhint %}
